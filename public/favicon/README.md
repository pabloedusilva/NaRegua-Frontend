# Favicon

Coloque aqui o arquivo `favicon.ico` da barbearia.

**Boas práticas:**
- Nome padrão: `favicon.ico` (32x32px ou multirresolução)
- Formatos modernos opcionais:
  - `favicon-16x16.png`
  - `favicon-32x32.png`
  - `apple-touch-icon.png` (180x180px para iOS)
  - `android-chrome-192x192.png` e `android-chrome-512x512.png` (para PWA)

**Uso:**
- Referenciado em `Client/index.html` via `<link rel="icon">`.
- Servido automaticamente em `/favicon.ico` pelo Vite (pasta `public`).
